---
title: NPM Scripts
---

## Dev

```bash
pnpm dev
```

It will run your `src/app/main.ts` code in watch mode.

## Build

```bash
pnpm build
```

Build `app` and `lib` into `dist/` folder

## Test

```bash
pnpm test
```

Unit testing using nodejs test runner by default.

## Lint

```bash
pnpm lint
```

Eslint code quality checks.

## Format

```bash
pnpm format
```

Prettier code format.

## Typecheck

```bash
pnpm typecheck
```

Typescript type check.

## Clean

```bash
pnpm clean
```

Remove `dist` and caches.

## Commit

```bash
pnpm commit
```

Commit with terminal UI.

## Pack

```bash
pnpm pack
```

Pack `dist` into tgz.
