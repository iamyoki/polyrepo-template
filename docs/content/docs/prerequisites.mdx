## Nodejs

A minimum version of Node.js 24 required.

## Corepack

Install corepack

```bash
npm install -g corepack
```

Enable corepack

```bash
corepack enable
```

## Pnpm

Install dependencies

```bash
pnpm install
```

<Callout title="Corepack will automatically install pnpm with a lock version.">
  You don't need to manually install pnpm
</Callout>
