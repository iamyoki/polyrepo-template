## Features

<Cards>

<Card icon={<AppWindow className="text-purple-300" />} title='App & Library'>

Develop both app and library under one repository, optionally distribute the library to npm or github packages regsitry.

</Card>

<Card icon={<CpuIcon className="text-blue-300" />} title='CI/CD Automation'>

Automatically release versions, generate CHANGELOG, and publish packages via github workflows.

</Card>

<Card icon={<Ruler className="text-green-300" />} title='Less config & Consistency'>

Minimalist, professional, and practical. All toolchain are set ready for you.

</Card>

<Card icon={<BookText className="text-yellow-300" />} title='Docs'>

Integrated Fumadocs to build your own beautiful docs site.

</Card>

</Cards>
