## Source Code of Package

`src/lib/index.ts` is your package source entry file, every module exported from will be published to npm/github packages registry.

```json tab="pacakge.json"
"exports": {
  ".": {
    "types": "./dist/lib/index.d.ts",
    "development": "./src/lib/index.ts",
    "import": "./dist/lib/index.js",
    "default": "./dist/lib/index.js"
  }
}
```
